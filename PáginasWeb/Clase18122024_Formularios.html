Formularios HTML

Los formularios en HTML, son elementos que nos van a dotar de cierta funcionalidad y dinamismo a nuestras paginas web. Nos vana a permitir interactuar con nuestras paginas web y por lo tanto son elementos fundamentales
para recoger informacion para posteriormente gestionarla.
Los formularios nos van a permitir dos tipos de funcionalidades principales:
  • Recoger informacion
  • Los formularios que se han guardado para posteriormente procesarlos o para poder realizar cualqier tipo de operacion que se requiera con ellos.

Los principales objetivos que tenemos que tener en cuenta a la hora de diseniar un formulario HTML,
son los siguientes:
  • El formulario tiene que ser los mas sencillos posible a la hora de insertar datos por parte del usuario
  • La experiencia del usuario/usabilidad(UX) tiene que ser los mas agradable posible
  • Los datos introducidos en los formularios, tiene que tener un formato predecible y esperado
    P.E Si vamos a introducir el nombre de usuario, facilitaremos al usuario la ubicacion donde debe colocarlo.
  • Reducir los errores al introducir los datos en la medidad de los posible y comunicar claramente al usuario
    si esta dando algun tipo de erros al introducirlos...

Etiquetas basicas para la creacion de un Formulario en html

Para crear un formulario en HTML, utilizaremos la etiqueta form con apertura y cierre. A parte de utilizar la etiqueta form, que establece que se va a usar/crear un formulario, tenemos que tener
en cuenta una serie de propiedades que vamos a usar con ella:
  • action: Indica la URL o BBDD, web, correo... (Direccion Backend) a la que queremos enviar
    la informacion de nuestro formulario.
  • method: Metodo puede ser get o post. Get se utiliza cuando queremos
    mostrar la indormacion que se envia de nuestro formulario a traves del protocolo HTTP/HTTPS de nuestras
    web.
    Post se utiliza cuando queremos ocultar la informacion que se envia de nuestro formulario a traves del
    protocolo HTTP/HTTPS de nuestras web.
  • name: se utiliza para dar un nombre al formulario
  • target: se utiliza para abrir el formulario en una pestamia nueva (target=_blank)
  • entype: codificacion de informacion mediante token.
  • autocomplete:Tiene una funcion on/off que sirve para activar y desactivar el autocompletado de
    nuestra web.

Tipos de datos que podemos gestionar en un Formulario de HTML


En un formluario, podemos gestionar cualquier tipo de informacion que se requiera. Pero los mas
utilizados o que vamos a ustilizar mas, son los siguientes:
  • Informacion de texto: Nombres, Apellidios, direcciones fisicas, etc. Para esto, utilizaremos
    la etiqueta input y textarea (input type=text y el textarea con etiqueta de apertura y cierre).
  • Numeros y cantidades: precios, edades, cuantias, etc.Utilizaremos el elemento input del tipo
    number (input type=number)
  • Fechas y horas: Fechas de nacimiento, calendarios, etc. Utilizaremos el input date, input week, etc
  • Opciones de verdadero o falso: Utilizaremos el tipo input type=checkbox
  • Opciones unicas: podremos elegir entre dos o mas opciones. input type = radio o la etiqueta select
  • Opciones unicas simples: donde utilizaremos el select multiple, input checkbox, etc
  • Opciones de listas abiertas: datalist
  • Seleccionar colores: input type=color
  • Seleccionas un archivo: input type=file

campos de Entrada de Informacion Numerica

Para poder obtener informacion numerica de nuestros formularios, podemos utilizar dos tipos de campos existentes:
  • input type=number con el cual estamos indicando que el valor de entrada sera numerico
  • campo de rango: input type=range
  • los atributos que van asociados a estos: min (establece un valor minimo para pasr la validacion)
    max (numero maximo permitido para pasar la validacion), el step (permite la validacion entre Numeros
    para pasar la validacion).

Campos de Entrada Numericos: fecha y hora

Dentro de los formularios de HTML, podemos uitilizar un tipo de input que nos permite estableces elementos
de fecha y hora
Entre ellos los mas comunes o que mas vamos a utilizar, podemos destacar los siguientes:
  • Fecha: utilizaremos el input = date
  • Hora: utilizaremos el input = time
  • Fecha y hora local: utilizaremos en input = datetime-local
  • Mes: Utilizaremos el input = month
  • Semana: utilizaremos em input = week
